## Douyin TV

一个轻量的 Android TV WebView 应用，用遥控器操作抖音网页版。

## 功能

- 全屏加载抖音推荐页
- 遥控器方向键映射为网页快捷键
- 长按确认键打开功能菜单
- 虚拟鼠标模式，用于登录、选择页面元素
- 页面加载失败时提供重试入口

## 遥控器操作

| 操作 | 行为 |
| --- | --- |
| 上 | 上一个视频 |
| 下 | 下一个视频 |
| 左 | 左侧动作 |
| 右 | 右侧动作 |
| 确认短按 | 打开评论 / 点击 |
| 确认长按 | 打开功能菜单 |
| 返回 | 关闭菜单 / 退出鼠标模式 / 返回页面 / 退出应用 |

## 功能菜单

- 鼠标模式
- 播放器全屏
- 网页沉浸式全屏
- 暂停 / 播放
- 刷新页面
- 自动连播开关
- 清屏
